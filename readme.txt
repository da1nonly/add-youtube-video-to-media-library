Add Youtube Video to Media Library.

== Description ==

Adds the ability to submit Youtube Videos to the Media Library. 

= Translations =

This plugin distributes with translations for the following language(s):

* English - default -

== Installation ==

1. Unpack the download-package
2. Upload folder include all files to the `/wp-content/plugins/` directory. 
3. Activate the plugin through the `Plugins` menu in WordPress
4. Add a video to with the "Add Video" link
